\doxysubsection{Detailed Description}
Internal indication of the error condition. 

Each error condition is specified by unique index from 0x00 up to 0x\+FF. Variable (from manufacturer section in the Object Dictionary) contains up to 0x\+FF bits (32bytes) for the identification of the specific error condition. (Type of the variable is C\+A\+Nopen O\+C\+T\+E\+T\+\_\+\+S\+T\+R\+I\+NG.)

If specific error occurs in the stack or in the application, \mbox{\hyperlink{group___c_o___emergency_ga9c9b7258698aadf6e9365bc0d1324be8}{C\+O\+\_\+error\+Report()}} sets specific bit in the {\itshape Error Status Bits} variable. If bit was already set, function returns without any action. Otherwise it prepares emergency message.

\mbox{\hyperlink{group___c_o___emergency_ga9c9b7258698aadf6e9365bc0d1324be8}{C\+O\+\_\+error\+Report()}}, \mbox{\hyperlink{group___c_o___emergency_ga14947a08d1c68446ee3356f00159174f}{C\+O\+\_\+error\+Reset()}} or \mbox{\hyperlink{group___c_o___emergency_ga0f165b478424ddb872dfdfafc9bcfdd0}{C\+O\+\_\+is\+Error()}} functions are called with unique index as an argument. (However \mbox{\hyperlink{group___c_o___emergency_ga9c9b7258698aadf6e9365bc0d1324be8}{C\+O\+\_\+error\+Report()}}, for example, may be used with the same index on multiple places in the code.)

Macros defined below are combination of two constants\+: index and \mbox{\hyperlink{group___c_o___e_m__error_codes}{C\+A\+Nopen Error codes}}. They represents specific error conditions. They are used as double argument for \mbox{\hyperlink{group___c_o___emergency_ga9c9b7258698aadf6e9365bc0d1324be8}{C\+O\+\_\+error\+Report()}}, \mbox{\hyperlink{group___c_o___emergency_ga14947a08d1c68446ee3356f00159174f}{C\+O\+\_\+error\+Reset()}} and \mbox{\hyperlink{group___c_o___emergency_ga0f165b478424ddb872dfdfafc9bcfdd0}{C\+O\+\_\+is\+Error()}} functions.

Stack uses first 6 bytes of the {\itshape Error Status Bits} variable. Device profile or application may define own macros for Error status bits using \mbox{\hyperlink{group___c_o___e_m__error_status_bits_ga643499974d43b6f758f8cd67718305e4}{C\+O\+\_\+\+E\+M\+\_\+\+M\+A\+N\+U\+F\+A\+C\+T\+U\+R\+E\+R\+\_\+\+S\+T\+A\+RT}} and \mbox{\hyperlink{group___c_o___e_m__error_status_bits_gadf273028f8a34935d32df9905448f9d6}{C\+O\+\_\+\+E\+M\+\_\+\+M\+A\+N\+U\+F\+A\+C\+T\+U\+R\+E\+R\+\_\+\+E\+ND}} values. Note that {\itshape Error Status Bits} must be large enough (up to 32 bytes).
