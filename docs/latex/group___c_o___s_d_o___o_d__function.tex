Optional application specific function, which may manipulate data downloaded or uploaded via S\+DO. 

Object dictionary function is external function defined by application or by other stack files. It may be registered for specific Object dictionary entry (with specific index). If it is registered, it is called (through function pointer) from S\+DO server. It may verify and manipulate data during S\+DO transfer. Object dictionary function can be registered by function \mbox{\hyperlink{group___c_o___s_d_o_gab433430abe125bb16c08e7be3e034985}{C\+O\+\_\+\+O\+D\+\_\+configure()}}.\hypertarget{group___c_o___s_d_o___o_d__function_autotoc_md28}{}\doxysubparagraph{S\+D\+O download (writing to Object dictionary)}\label{group___c_o___s_d_o___o_d__function_autotoc_md28}
After S\+DO client transfers data to the server, data are stored in internal buffer. If data contains multibyte variable and processor is big endian, then data bytes are swapped. Object dictionary function is called if registered. Data may be verified and manipulated inside that function. After function exits, data are copied to location as specified in \mbox{\hyperlink{struct_c_o___o_d__entry__t}{C\+O\+\_\+\+O\+D\+\_\+entry\+\_\+t}}.\hypertarget{group___c_o___s_d_o___o_d__function_autotoc_md29}{}\doxysubparagraph{S\+D\+O upload (reading from Object dictionary)}\label{group___c_o___s_d_o___o_d__function_autotoc_md29}
Before start of S\+DO upload, data are read from Object dictionary into internal buffer. If necessary, bytes are swapped. Object dictionary function is called if registered. Data may be manipulated inside that function. After function exits, data are transferred via S\+DO server.\hypertarget{group___c_o___s_d_o___o_d__function_autotoc_md30}{}\doxysubparagraph{Domain data type}\label{group___c_o___s_d_o___o_d__function_autotoc_md30}
If data type is domain, then length is not specified by Object dictionary. In that case Object dictionary function must be used. In case of download it must store the data in own location. In case of upload it must write the data (maximum size is specified by length) into data buffer and specify actual length. With domain data type it is possible to transfer data, which are longer than \mbox{\hyperlink{group___c_o___s_d_o_ga93b6bae3c759f44efab5ceeae57d4ac4}{C\+O\+\_\+\+S\+D\+O\+\_\+\+B\+U\+F\+F\+E\+R\+\_\+\+S\+I\+ZE}}. In that case Object dictionary function is called multiple times between S\+DO transfer.\hypertarget{group___c_o___s_d_o___o_d__function_autotoc_md31}{}\doxysubparagraph{Parameter to function\+:}\label{group___c_o___s_d_o___o_d__function_autotoc_md31}
