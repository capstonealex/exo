\doxysubsection{Detailed Description}
Object contains all information about the object being transferred by S\+DO server.

Object is used as an argument to \mbox{\hyperlink{group___c_o___s_d_o___o_d__function}{Object Dictionary function}}. It is also part of the \mbox{\hyperlink{struct_c_o___s_d_o__t}{C\+O\+\_\+\+S\+D\+O\+\_\+t}} object.
