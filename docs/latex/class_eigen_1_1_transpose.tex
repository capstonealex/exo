Expression of the transpose of a matrix. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Matrix\+Type} & the type of the object of which we are taking the transpose\\
\hline
\end{DoxyTemplParams}

It is the return type of \mbox{\hyperlink{class_eigen_1_1_dense_base_ac575a73f01b20b628bb910d0f395cb33}{Matrix\+Base\+::transpose()}} and \mbox{\hyperlink{class_eigen_1_1_matrix_base_afacca1f88da57e5cd87dd07c8ff926bb}{Matrix\+Base\+::adjoint()}} and most of the time this is the only way it is used.
