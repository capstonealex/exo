\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename View\+Op, typename Matrix\+Type$>$\newline
class Eigen\+::\+Cwise\+Unary\+View$<$ View\+Op, Matrix\+Type $>$}

Generic lvalue expression of a coefficient-\/wise unary operator of a matrix or a vector. 


\begin{DoxyTemplParams}{Template Parameters}
{\em View\+Op} & template functor implementing the view \\
\hline
{\em Matrix\+Type} & the type of the matrix we are applying the unary operator\\
\hline
\end{DoxyTemplParams}
This class represents a lvalue expression of a generic unary view operator of a matrix or a vector. It is the return type of \mbox{\hyperlink{_common_cwise_unary_ops_8h_a560b678a662ccfce4da582595899a0b7}{real()}} and \mbox{\hyperlink{_common_cwise_unary_ops_8h_a8cc3d2f4413a156b2ec6a3097efd3aa4}{imag()}}, and most of the time this is the only way it is used.
